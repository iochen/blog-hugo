  • Linux下使用ExifTool对EXIF的操作

     2020-02-04 |  Tutorials  

    前言 EXIF数据可以包括你的相机信息，快门，光圈，ISO，创建时间，编辑信息，GPS信息等信息，他们会在无心之中泄漏你的隐私，特别是上传照片到互联网上时，那更是有极大的泄漏风险。因此，在上传照片到网路上时，请务必删除EXIF信息。
    本篇文章就是要教你在Linux下，如何使用ExifTool查看、修改与删除图片的EXIF数据
    安装ExifTool openSUSE zypper in exiftool  Ubuntu apt-get install libimage-exiftool-perl  CentOS yum install perl-Image-ExifTool  Others 到 ExifTool官网 下载二进制包，解压并进入。如有需要，请链接到/usr/bin/

     -a 查看所有metadata  导出为json exiftool -j {{InputFile}} > {{OutputFile}}  可选参数：
