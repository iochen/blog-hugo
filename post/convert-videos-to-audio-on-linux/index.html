Linux下使用FFmpeg,将mp4等格式转mp3格式

 2019-08-18 |  Apps   FFmpeg  

本文章将介绍 Linux下使用FFmpeg,将各类格式互相转换，以及如何批量转换

单个转换

命令

ffmpeg -i {输入文件} {输出文件}

示例

mp4 转 mp3

ffmpeg -i foo.mp4 foobar.mp3

flv 转 mp3

ffmpeg -i foo.flv foobar.mp3

mp4 转 wav

ffmpeg -i foo.mp4 foobar.wav

wav 转 mp3

ffmpeg -i foo.wav foobar.mp3

批量转换

比如，文件夹下有*.mp4文件，要批量转换为XX.mp3文件，可以用如下方法

for i in ./*.mp4
do
ffmpeg -i $i ${i}.mp3
done

再比如，文件夹下所有文件文件，都要批量转换为XX.mp3文件，可以用如下方法

for i in ./*
do
ffmpeg -i $i ${i}.mp3
done
